Getting Started

Requirements for Host

  • Internet connectivity
  • Linux (we recommend Ubuntu 16.04)
  • Docker
  • 1 GiB RAM
  • 1 GiB Storage

Installation Instructions

Before installing Hostero, you first need to install Docker. We recommand to install Docker using our instructions, but you can install it using alternative ways as well.

1. Install Docker

Run this command to install it:

curl https://releases.rancher.com/install-docker/17.12.sh | sh

Note: You need to have curl installed. If it's not installed, run: apt install curl

2. Install Hostero

Run this command to install it:

sudo docker run -e CATTLE_HOST_LABELS='account=ACCOUNT_NAME' --rm --privileged -v /var/run/docker.sock:/var/run/docker.sock -v /var/lib/rancher:/var/lib/rancher rancher/agent:v1.2.9 http://infra.hostero.eu:8080/v1/scripts/2E28C27EE87F5E1A6B7F:1514678400000:ShibdGaXFOlmN3QGGHj1UFuP1lU

Note: Replace ACCOUNT_NAME with the variable from your account.
