Requirements

Please make sure your device meets all the following requirements:

  • Internet connection
  • MacOS Sierra 10.12 or newer release
  • Docker 17.06 (all supported versions)
  • 4 GB Storage
  • 4 GB RAM

Note: This setup is experimental, support for installing the MacOS miner is limited.

Instructions

Hostero requires Docker to work. We recommend to install Docker via our link, but you can install it using alternative ways as well.

1. Install Docker

Download the installer and proceed to install Docker as any other software. You can find further instructions on the official Docker website.

Get Docker Community Edition 17.06.2-ce-mac27 2017-09-06

Important: Do not update Docker to the latest version, the miner will not work anymore.

2. Install Hostero

Run this command in the Terminal:
{{#account_id}}
docker run -e CATTLE_HOST_LABELS="account={{account_id}}" --rm --privileged -v /var/run/docker.sock:/var/run/docker.sock rancher/agent:v1.2.9 http://europe1.hostero.eu:8080/v1/scripts/2E28C27EE87F5E1A6B7F:1514678400000:ShibdGaXFOlmN3QGGHj1UFuP1lU
