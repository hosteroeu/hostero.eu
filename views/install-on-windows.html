Requirements

Please make sure your device meets all the following requirements:

  • Internet connection
  • Windows 10 64bit (Pro, Enterprise or Education)
  • Docker 17.06
  • 4 GB Storage
  • 4 GB RAM

Instructions

Hostero requires Docker to work. We recommend to install Docker via our link, but you can install it using alternative ways as well (supported versions).

1. Install Docker

Download the installer and proceed to install Docker as any other software. You can find further instructions on the official Docker website.

Get Docker Community Edition 17.06.2-ce-win27 2017-09-06

2. Install Hostero

Run this command in PowerShell:
{{#account_id}} docker run -e CATTLE_HOST_LABELS="account={{account_id}}" --rm --privileged -v /var/run/docker.sock:/var/run/docker.sock -v /var/lib/rancher:/var/lib/rancher rancher/agent:v1.2.9 http://infra.hostero.eu:8080/v1/scripts/2E28C27EE87F5E1A6B7F:1514678400000:ShibdGaXFOlmN3QGGHj1UFuP1lU {{/account_id}} {{^account_id}} docker run -e CATTLE_HOST_LABELS="account=ACCOUNT_ID" --rm --privileged -v /var/run/docker.sock:/var/run/docker.sock -v /var/lib/rancher:/var/lib/rancher rancher/agent:v1.2.9 http://infra.hostero.eu:8080/v1/scripts/2E28C27EE87F5E1A6B7F:1514678400000:ShibdGaXFOlmN3QGGHj1UFuP1lU {{/account_id}}
